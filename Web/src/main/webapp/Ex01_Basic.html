1. Front 개발 : 메모장, 웹 브라우저
2. 개발 툴 : VSCode(프론트 개발에 주로 사용) - Live Server, atom, 이클립스 등등
3. Front + Server(서버는 웹서버, WAS 2개로 나눠짐

4. 크롬 개발자도구(F12) : 디버깅, 네트워크 정보, 소스정보
5. 웹 표준 : HTML5(브라우저끼리 통일, HTML5로 통일
브라우저마다 해석기(엔진, 파서가)이 다르다.
근데 HTML5로 해석 시키게 표준을 정해서 통일 시켰다는 의미
ex)크롬, 엣지, 사파리, 파이어폭스 등 -> 개발자는 HTML5 기준으로 만들면 된다
